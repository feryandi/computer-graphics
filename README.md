# Grafika1
Tugas 1 Grafika, membuat animasi sederhana dengan direct access ke frame buffer.

1. compile --> 
gcc -o grafika1 grafika1.c

2. pindah layar dulu --> 
control-alt-F2

3. login --> 
login: username (bisa dicek dengan echo $USER)
password: akucantik (gadeng, password lu apa)

4. run program --> 
sudo ./grafika1

5. senyum bahagia
